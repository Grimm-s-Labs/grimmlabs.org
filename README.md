# Grimm Labs

Official website for Grimm Labs

## Overview

This site is built with **Hugo**, utilizing a custom raw HTML/CSS theme.
## Development

### Prerequisites
*   [Hugo](https://gohugo.io/) (Extended version recommended)

### Running Locally
```bash
hugo server
```
Access the site at `http://localhost:1313`.

### Managing Content

Please refer to [GUIDE.md](GUIDE.md) for detailed instructions on:
*   Adding new Team members.
*   Writing new Blog posts.
*   Managing assets and images.

## Structure
*   `content/`: Markdown content for pages, posts, and team profiles.
*   `layouts/`: Custom HTML templates.
*   `static/`: CSS, images, and other static assets.
*   `hugo.toml`: Main configuration file.
